# Code of Conduct

This project adopts the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

The full text is canonical at the link above. A summary:

- **Treat others with respect.** Assume good faith, give constructive feedback, accept it gracefully.
- **Keep discussion technical.** Focus on the work, not the person.
- **Disagreement is welcome; hostility is not.** Strong opinions are fine; personal attacks, harassment, and demeaning language are not.
- **Maintainers will enforce these standards** in issues, pull requests, discussions, and any other project space.

## Scope

This Code of Conduct applies to all project spaces: the GitHub repository (issues, pull requests, discussions, commit messages), and any future official communication channels.

## Reporting

If you experience or witness unacceptable behavior, report it privately to the maintainer via the contact in the repository's GitHub profile, with subject line `[sleipnir-conduct]`.

All reports will be reviewed and investigated promptly and fairly. Maintainers are obligated to respect the privacy of the reporter.

## Enforcement

Maintainers may take any action they deem appropriate, up to and including a permanent ban from the project, in response to behavior they judge inappropriate. The full enforcement guidelines from the Contributor Covenant 2.1 apply.
